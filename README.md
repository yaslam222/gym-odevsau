# SporSalonuProjesi

Spor Salonu Yönetim ve Randevu Sistemi



Bu proje Web Programlama Dersi için yapılmaktadır.

Amaç; spor salonları için üyelik, antrenör, hizmet ve randevu yönetimi sağlayan modern bir web uygulaması oluşturmaktır.



Uygulama; ASP.NET Core MVC, Entity Framework Core, SQL Server, Identity yapısı, REST API ve yapay zekâ tabanlı öneri sistemi kullanılarak geliştirilecektir.
